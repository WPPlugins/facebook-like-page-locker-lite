Facebook Likebox Popup | Facebook Like Countdown Locker | Lock Page & Post with Facebook Likebox

== Description ==

If you want to convert your site traffic to your <strong>Facebook</strong> fan page likes and boost your fan page social media profile, then <strong>Facebook Like Page Locker Lite</strong> is the right one !

<strong>Facebook Like Page Locker Lite</strong> is the <strong>Free</strong> WordPress Plugin to lock any of your pages with a <strong>Facebook Fan Page Like Popup</strong> until the site visitors likes your fan page. Use it as just the Facebook Likebox Popup with close button or Facebook Likebox Popup Locker without close button, choice is yours.


= See full features list below: =
* Facebook Likebox Popup
    - Lock page or just use it as likebox popup
* Two locker methods available
    - Like Button Lock
    - Like Box Lock
*   Configure each and every section of locker
    - Customize the heading of the locker
    - Configure to show either Like Button or Like Box with all individual settings provided by Facebook
    - Enable or disable countdown counter
    - Customize Unlock Message
    - Enable or disable close action
    - Mobile Friendly

* Instant Lock Popup close after page like by site visitor
* User Friendly Interface with proper preview of Like box and Like button


    

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/facebook-like-page-locker-lite` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Facebook Like Page Locker Lite settings page to configure the plugin



== Frequently Asked Questions ==

= Can I use this plugin just as Facebook Likebox Popup ? =

Yes you can by just enabling the close action in the plugin locker settings

= How will it prevent from displaying the locker multiple times for same user ? =

The plugin will set the cookie as soon as a user likes the configured page and same cookie will be checked before displaying the locker again.
